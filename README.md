# Healthcare Hack NYC

AI Healthcare Hack NYC — hosted by Arya Health + localhost:nyc, sponsored by Twilio AI Startup Searchlight. July 11, 2026 · 307 W 36th St, floor 13, NYC.

## The Challenge

Build a production-ready **voice/text AI agent** that carries a full conversation end-to-end, grounded in domain knowledge and personalized to each caller. Reliability, guardrails, security, and scalability are prerequisites, not nice-to-haves.

**Twilio is mandatory** for the telephony layer to qualify for sponsor prizes.

## Suggested Stack Checklist

- [ ] **Telephony** — Twilio (required for prizes)
- [ ] **Agent framework** — orchestrate/modularize conversation logic
- [ ] **Voice models** — TTS/STT
- [ ] **Knowledge base** — external sources for grounded context
- [ ] **Caller info** — recognize and personalize per caller
